PPE-Violation-Detection

PPE Violation Detection System uses existing IP cameras to identify if specific individuals to be monitored are wearing PPE kit or not. It generates notifications and sends it to recognized individuals or concerned authorities to warn if anyone is found violating PPE compliance.

Project Overview

This is Machine Learning Project

  • First We collect the data set in many form and every angle
  • We use Roboflow for the Data annotation
  • We use DGX 100 for training the our model
  • We use yolov7 new state of the art real time object detection
  • Jupyter use for the Linux terminal where the DGX 100 trainthe model
  • Gmail Api we use for sending the mail
